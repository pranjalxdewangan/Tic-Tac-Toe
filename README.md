This is a console-based Tic Tac Toe game developed in C++ as an initial programming project.
The project focuses on understanding basic game logic, control flow, and functions in C++.

Project Objective

Learn how to build a complete game using C++
Practice arrays, loops, and conditional statements
Implement win and draw conditions
Understand turn-based gameplay logic

Technologies Used

Language: C++
Concepts:
Arrays
Loops
Functions
Conditional statements
Basic input/output
